## 传感器

### 创建传感器

创建传感器实例有两种方式，具体如下所示。

1. 根据传感器参数，创建传感器实例。

   - 方法原型：
   
     ```cpp
     static std::shared_ptr<SensorInterface> create_sensor(
         const std::unordered_map<std::string, std::string>& params)
     ```
   
   - 参数说明：
   
     <table>
     <thead>
     <tr>
     <th><strong>参数名称</strong></th>
     <th><strong>类型</strong></th>
     <th><strong>说明</strong></th>
     </tr>
     </thead>
     <tbody>
     <tr>
     <td>
     
     `params`
     
     </td>
     <td>
     
     输入参数
     
     </td>
     <td>
     
     传感器相关参数，例如：
     
     ```json
     {
        "sensor_name":"rs1", #传感器名称
        "sensor_type":"realsense", #传感器厂商
        "serial_number":"215322070063", #序列号
        "color_width":"640", #彩色图宽
        "color_height":"480", #彩色图高
        "depth_width":"640", #深度图宽
        "depth_height":"480", #深度图高
        "align_to_color":"true", #是否将深度图对齐到彩色图
        "frame_rate":"30" #帧率
     }
     ```
     
     </td>
     </tr>
     </tbody>
     </table>  
   
   - 返回值：
   
     若创建成功，则返回 `SensorInterface` 类型的共享智能指针；否则返回 `nullptr`。

2. 根据配置文件，创建传感器实例。

   - 方法原型：
   
     ```cpp

         const std::string& config_path)
     ```
   
   - 参数说明：

     <tbody>
     <tr>
     <td>

     </td>
     <td>
     
     输入参数

     <td>
     
     配置文件路径。配置文件为 JSON 文件，例如：
     
     ```json
     {
        "sensor_name":"rs1", #传感器名称
        "sensor_type":"realsense", #传感器厂商
        "serial_number":"215322070063", #序列号
        "color_width":"640", #彩色图宽
        "color_height":"480", #彩色图高
        "depth_width":"640", #深度图宽
        "depth_height":"480", #深度图高
        "align_to_color":"true", #是否将深度图对齐到彩色图
        "frame_rate":"30" #帧率
     }
     ```

   - 返回值：
   
     若创建成功，则返回 `SensorInterface` 类型的共享智能指针；否则返回 `nullptr`。

### 连接传感器

- 方法原型：

  ```cpp
  AbcErrorCode connect()
  ```

- 返回值：

  `AbcErrorCode::SUCCESS` 表示成功；其他错误码，请参见 [AbcErrorCode](/sdk/c-plus-plus/data-structures#AbcErrorCode)。

### 断开传感器

- 方法原型：

  ```cpp
  AbcErrorCode disconnect()
  ```

- 返回值：

  `AbcErrorCode::SUCCESS` 表示成功；其他错误码，请参见 [AbcErrorCode](/sdk/c-plus-plus/data-structures#AbcErrorCode)。

### 注册传感器数据处理回调函数

此方法用于注册一个回调函数，当传感器采集到新数据时，系统会自动调用该回调函数进行数据处理。

- 方法原型：

  ```cpp
  void set_data_callback(SensorCallback cb)
  ```

- 参数说明：

  | 参数名称 | 类型 | 说明 |
  |:---|:---|:---|
  | `cb` | 输入参数 | 当传感器数据更新时，系统会自动调用此回调函数。`SensorCallback` 为函数指针类型。其数据类型，请参见 [SensorCallback](/sdk/c-plus-plus/data-structures#SensorCallback)。 |

- 返回值：

  无。
