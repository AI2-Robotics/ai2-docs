## SensorFactory {#SensorFactory}

`SensorFactory` 类用于创建传感器和深度相机实例。

### 创建传感器实例

1. 根据传感器参数，创建传感器实例。

   - 方法原型：

     ```python
     create_sensor(params: dict[str, str]) -> SensorInterface
     ```

   - 参数说明：

     <table>
        <tbody>
           <tr>
              <td>**参数名称**</td>
              <td>**说明**</td>
           </tr>
           <tr>
              <td>`params`</td>
              <td>
                   传感器相关参数，例如：
                               
                    ```json
                   {
                        "sensor_name":"rs1", # 传感器名称
                        "sensor_type":"realsense", # 传感器厂商
                        "serial_number":"215322070063", # 序列号
                        "color_width":"640", #彩色图宽
                        "color_height":"480", # 彩色图高
                        "depth_width":"640", # 深度图宽
                        "depth_height":"480", # 深度图高
                        "align_to_color":"true", # 是否将深度图对齐到彩色图
                        "frame_rate":"30" # 帧率
                   }
                    ``` 
              </td>
           </tr>
        </tbody>
     </table>

   - 返回值：

     若创建成功，则返回 [SensorInterface](#SensorInterface) 实例；否则返回 `None`。

2. 根据配置文件，创建传感器实例。

   - 方法原型：

     ```python

     ```

   - 参数说明：

              <td>**说明**</td>
           </tr>
           <tr>
              <td>`config_path`</td>
              <td>
                   配置文件路径。配置文件为 JSON 文件，例如：
                               
                    ```json
                   {
                        "sensor_name":"rs1", # 传感器名称
                        "sensor_type":"realsense", # 传感器厂商
                        "serial_number":"215322070063", # 序列号
                        "color_width":"640", #彩色图宽
                        "color_height":"480", # 彩色图高
                        "depth_width":"640", # 深度图宽
                        "depth_height":"480", # 深度图高
                        "align_to_color":"true", # 是否将深度图对齐到彩色图
                        "frame_rate":"30" # 帧率
                   }
                    ``` 
              </td>
           </tr>
        </tbody>
     </table>

   - 返回值：

     若创建成功，则返回 [SensorInterface](#SensorInterface) 实例；否则返回 `None`。

## SensorInterface {#SensorInterface}

`SensorInterface` 类提供了一系列用于控制传感器的公共方法。在使用前，请确保已创建传感器实例。

### 属性

| 属性（只读） | 数据类型 | 说明 |
|:---|:---|:---|
| `name` | str | 传感器名称 |
| `status` | [SensorStatus](/sdk/python/data-structures#SensorStatus) | 传感器状态 |
| `type` | [SensorType](/sdk/python/data-structures#SensorTypeSensorType) | 传感器类型 |

### 连接传感器

在创建传感器实例后，用户可通过如下方法与传感器建立连接。

- 方法原型：

  ```python
  connect(self) -> alphabot.core.AbcErrorCode
  ```

- 返回值：

  `AbcErrorCode.SUCCESS` 表示成功；其他错误码，请参见 [AbcErrorCode](/sdk/python/data-structures#AbcErrorCode)。

### 断开传感器

用户可通过如下方法断开与传感器的连接。

- 方法原型：

  ```python
  disconnect(self) -> alphabot.core.AbcErrorCode
  ```

- 返回值：

  `AbcErrorCode.SUCCESS` 表示成功；其他错误码，请参见 [AbcErrorCode](/sdk/python/data-structures#AbcErrorCode)。

## SyncPolicy {#SyncPolicy}

`SyncPolicy` 类是一个枚举器，用于表示同步策略。

<Aside>
此类为枚举类，用户应直接使用类中定义的常量成员，无需自行实例化。
</Aside>

| 常量成员 | 值 | 说明 |
|:---|:---|:---|
| `NEAREST` | 0 | 最近时间戳匹配 |
| `INTERPOLATE` | 1 | 插值匹配 |
| `EXTRAPOLATE` | 2 | 外推匹配 |

## SyncMode {#SyncMode}

`SyncMode` 类是一个枚举器，用于表示同步模式。

<Aside>
此类为枚举类，用户应直接使用类中定义的常量成员，无需自行实例化。
</Aside>

| 常量成员 | 值 | 说明 |
|:---|:---|:---|
| `LATEST_TIMESTAMP` | 0 | 基于最新时间戳同步（默认模式） |
| `REFERENCE_SENSOR` | 1 | 基于参考传感器同步 |

| `active_pull` | bool |  是否采用主动拉流模式。默认为 `false`（后台自动拉流）；设置为 `true` 时，仅在调用 `get_data()` 时拉取一帧，且不支持回调机制。|

## AbcIntrinsic {#AbcIntrinsic}

`AbcIntrinsic` 类用于表示相机内参数据，包含以下属性。

| 结构体成员 | 数据类型 | 说明 |
|:---|:---|:---|
| `width` | int | 图像宽度，单位为像素。 |
| `height` | int | 图像高度，单位为像素。 |
| `ppx` | float | 图像主点的水平坐标，以距左边缘的像素偏移量表示。 |
| `ppy` | float | 图像主点的垂直坐标，以距上边缘的像素偏移量表示。 |
| `fx` | float | 水平焦距，以像素宽度倍数表示。 |
| `fy` | float | 垂直焦距，以像素高度倍数表示。 |
| `coeffs` | list[float] | 畸变系数，列表长度固定为 5。各模型的参数顺序如下。<br/>- Brown-Conrady 模型：`[k1, k2, p1, p2, k3]`。<br/>- F-Theta 鱼眼镜头模型：`[k1, k2, k3, k4, 0]`。<br/>- 其他模型：参见其具体解释规则。 |


- 返回值：

  `True` 表示深度图像已对齐到彩色图像，`False` 表示深度图像未对齐到彩色图像。

### 注册深度相机数据处理回调函数

- 方法原型：

  ```python
  set_data_callback(self, arg0: typing.Callable) -> None
  ```

- 参数说明：

  | 参数名称 | 说明 |
  |:---|:---|
  | `arg0` | 当深度相机数据更新时，系统会自动调用此回调函数。回调函数接收类型为 [RGBDData](#RGBDData) 的参数。 |

  `None`。
