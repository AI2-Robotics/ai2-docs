Mastering the following operations will help you use the VR Teleoperation Device more efficiently.

## Starting the HMI Interface

ROBOT-CONTROL-HMI (hereinafter referred to as HMI) is the graphical user interface for controlling and monitoring the robot system. It serves as the core tool for interacting with the robot when using the VR Teleoperation Device. The HMI consists of two separate programs: the user-side program and the robot-side program. Both must be started before use, following the procedures below.

### Starting the User-Side Program

import operation_guide_1 from 'img/operation_guide_1.png';

<Steps>

1. On the industrial PC, download the [Microsoft Edge browser installation package](https://www.microsoft.com/en-us/edge/download).

2. In the folder containing the installer package, open a terminal and run the following command to install the browser.


   ```bash
   sudo dpkg -i microsoft-edge-stable_xxx_arm64.deb
   ```
   <Aside>
   Replace `microsoft-edge-stable_xxx_arm64.deb` in the command with the actual filename of the installation package.
   </Aside>


3. In the Ubuntu operating system, navigate to `Settings -> Default Applications -> Web (W)` , select Microsoft Edge, and set Microsoft Edge as the default browser.

4. On the industrial PC, open a terminal and enter the following commands.

   ```bash
   cd ~
   bash ~/user_docker_start.sh
   ```

5. After the script executes, the Microsoft Edge browser will launch the HMI interface automatically. If it does not open automatically, manually enter `http://127.0.0.1:4090` in the browser's address bar.

   <Image src={operation_guide_1} alt="" />

   <Aside>
   - The Exoskeleton Device, VR Control Device, and Active Exoskeleton Device are three independent systems. Therefore, when switching between starting the Exoskeleton Device, VR Control Device, and Active Exoskeleton Device, you must close all controls running under the previous device first. Ensure all buttons in the Robot Control panel are blue before switching devices.
   - To close the HMI, first press `Ctrl+C` to terminate both the user-side and robot-side programs, then close the corresponding terminals, and finally close the Microsoft Edge browser.
   </Aside>
       

</Steps>

### Starting the Robot-Side Program

<Steps>

1. On the industrial PC, open a new terminal window and remotely connect to the robot terminal using the `ssh` command.

   ```bash
   zpf@zpf-MIC-770V3H:~$ ssh robot@192.168.1.100
   # Password: robot
   robot@192.168.1.100's password: 
   # Connection is successful if the terminal username starts with "robot".
   robot@ter30jb3-ubuntu:~$ 
   ```

2. In the successfully connected terminal, enter the following commands.

   ```bash
   cd ~
   bash ~/robot_docker_start.sh
   ```
   <Aside>
   To close the HMI, first press `Ctrl+C` to terminate both the user-side and robot-side programs, then close the corresponding terminals, and finally close the Microsoft Edge browser.
   </Aside>
</Steps>

After Starting the HMI interface, you can [use the VR device](#using-vr-device) to control the robot.

### Starting the VR Control Device{#start-vr-control-device}

<Steps>
1. After ensuring that all [prerequisites](#preconfiguration) have been completed, click the `Start VR Control Device` button in the HMI interface. Upon successful startup, the button will turn red and the text will change to `Stop VR Control Device`.
   <Image src={operation_guide_8} alt="" />

   <Aside>
   - The Exoskeleton Device, VR Control Device, and Active Exoskeleton Device are three independent systems. Therefore, when switching between starting the Exoskeleton Device, VR Control Device, and Active Exoskeleton Device, you must close all controls running under the previous device first. Ensure all buttons in the Robot Control panel are blue before switching devices.
   - If the robot malfunctions or the robotic arm moves abnormally or becomes uncontrollable, click the `Stop VR Control Device` button immediately. The robot will emergency stop and cease response.
   </Aside>

2. Check the `System Status` panel. If `VR Device Status` displays `Data Normal, 3 entries`, the startup is successful.
</Steps>

## Data Acquisition

After [starting the VR Control Device](#start-vr-control-device), you can acquire data from the VR Teleoperation Device using the `Robot Data Acquisition` function in the HMI interface.

<Aside>
The Exoskeleton Device, VR Control Device, and Active Exoskeleton Device are three independent systems; do not mix them.
</Aside>

After successfully [creating a task](#create-task), you can start acquiring data.

   - Click the `Start Acquisition` button. 
      <Image src={operation_guide_42} alt="" />

   - Press the middle button of the foot pedal.

   <Image src={operation_guide_44} alt="" />

### Canceling Acquisition

import operation_guide_45 from 'img/operation_guide_45.png';
import operation_guide_46 from 'img/operation_guide_46.png';
import operation_guide_47 from 'img/operation_guide_47.png';

During the acquisition process, you can cancel acquisition at any time.

<Steps>
1. Use any of the following three methods to cancel data acquisition.

   - Click the `Cancel Acquisition` button. 
      <Image src={operation_guide_45} alt="" />
      A prompt message as shown below will pop up on the page. Click the blank area outside the pop-up window to close it.
      <Image src={operation_guide_46} alt="" />
   - Press the `Ctrl+4` keyboard shortcut.
   - Press the right button of the foot pedal.

2. After acquisition is canceled, the `Acquisition Status` in the `Robot Data Acquisition Status` panel will change from `Acquiring` to `Waiting to Start`. The running time will stop counting, and the acquisition time will reset to zero.
   <Image src={operation_guide_47} alt="" />
</Steps>

### Ending Acquisition

import operation_guide_48 from 'img/operation_guide_48.png';
import operation_guide_49 from 'img/operation_guide_49.png';
import operation_guide_50 from 'img/operation_guide_50.png';

After data acquisition is complete, you need to manually end the acquisition process.

<Steps>
1. Use any of the following three methods to end data acquisition.

   - Click the `End Acquisition` button.
      <Image src={operation_guide_48} alt="" />
      A prompt message as shown below will pop up on the page. Click the blank area outside the pop-up window to close it.
      <Image src={operation_guide_49} alt="" />
   - Press the `Ctrl+3` keyboard shortcut.
   - Press the middle button of the foot pedal.

2. After acquisition ends, the `Acquisition Status` in the `Robot Data Acquisition Status` panel will change from `Acquiring` to `Waiting to Start`. The running time will stop counting, the acquisition time will reset to zero, one data entry will be written to disk, and the acquisition entry count will increment by 1.
    <Image src={operation_guide_50} alt="" />

</Steps>
