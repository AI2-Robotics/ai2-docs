Welcome to using the Exoskeleton Teleoperation Device! This document guides you through the initial steps, including unboxing and connection, to prepare you for subsequent use.

<Aside type="caution">
Before you begin, please carefully read the [Safety Regulations](/en/exoskeleton-teleoperation/safety-regulations) and [Disclaimer](/en/statement/disclaimer).
</Aside>

## Unboxing

After receiving the Exoskeleton Teleoperation Device, follow the steps below to unbox it:

<Steps>

1. Inspect the packaging box to ensure it is intact.
   <Aside>
    If the box is damaged, please contact our company and the logistics company immediately.
   </Aside>

2. Refer to the list below to ensure no parts are missing or damaged.

   <Aside>
   - The following is for reference only; please refer to the actual delivered items.
   - Please keep the packaging box and materials properly for future transportation or maintenance.
   </Aside>

   |Item|Quantity|Item|Quantity|
   |---|---|---|---|
   |Left Teleoperation Arm|1|  Industrial PC|1|
   |Right Teleoperation Arm|1|  Monitor|1|
   |Mobile Stand|1|        Keyboard & Mouse|1|
   |Counterweight|1|          Foot Pedal|1|
   |Storage Basket|1|          Fastener Kit|1|
   |Telescopic Hook Balancer|4|   Tool Kit|1|

</Steps>

Next, follow the steps below to complete the assembly.

<Steps>
1. Install the four casters onto the chassis.

   <Image src={getting_started_1} alt="" />

2. Install the support bar onto the chassis.

   <Image src={getting_started_2} alt="" />

3. Using M6*35 screws, install the 600mm aluminum profiles (side connecting bars) and the teleoperation arm mounting base onto the support bar.

   <Image src={getting_started_3} alt="" width="400" />   

4. First, install the 540mm aluminum profiles (top side bars) onto the 600mm aluminum profiles (side connecting bars). Then, install the 800mm and 1550mm aluminum profiles (front and rear top load-bearing bars) onto the front and rear ends of the 540mm aluminum profiles (top side bars), respectively.

   <Image src={getting_started_4} alt="" />

5. First, install the adapter flange onto the teleoperation arm mounting base. Then, install the 460mm aluminum profile (arm load-bearing bar) onto the adapter flange. Finally, install the bumper end caps onto both ends of the 460mm aluminum profile.

   <Image src={getting_started_5} alt="" />

6. Install the left and right teleoperation arms onto the arm load-bearing bar.

   <Image src={getting_started_6} alt="" width="400"/>

7. First, install angle brackets onto both ends of the front and rear top load-bearing bars. Then, use cable ties to secure the four telescopic hook balancers to the front and rear load-bearing bars (excess cable tie length may be trimmed). Finally, attach the suspension hooks to the teleoperation arms.

   <Image src={getting_started_7} alt=""/>
   <Image src={getting_started_8} alt=""/>

8. Fix the storage basket at the bottom of the support bar and place the counterweight inside to maintain stand stability.

   <Image src={getting_started_9} alt="" />

9. The assembly of the exoskeleton teleoperation device is now complete.

   <Image src={getting_started_10} alt="" width="400"/>

</Steps>
