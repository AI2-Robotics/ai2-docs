在使用外骨骼遥操设备时，掌握以下操作将有助于您更高效地使用和操作设备。

## 启动 HMI 界面{#start-hmi-interface}

ROBOT-CONTROL-HMI（简称 HMI）是用于控制和监控机器人系统的图形用户界面，也是使用外骨骼遥操设备时与机器人交互的核心工具。HMI 分为用户端（user）与机器人端（robot）两部分程序，使用前需分别启动，具体操作如下。

### 启动用户端程序

import operation_guide_1 from 'img/operation_guide_1.png';

<Steps>

1. 在工控机上，下载 [Microsoft Edge 浏览器安装包](https://www.microsoft.com/zh-cn/edge/download?form=MA13FJ)。

2. 在安装包所在文件夹下，打开终端，通过如下命令进行安装。


   ```bash
   sudo dpkg -i microsoft-edge-stable_xxx_arm64.deb
   ```
   <Aside>
   请将命令中的 `microsoft-edge-stable_xxx_arm64.deb` 替换为实际的安装包文件名。
   </Aside>


3. 在 Ubuntu 操作系统中，依次点击 `设置 -> 默认应用程序 -> 网络(W)`，选择 Microsoft Edge，将 Microsoft Edge 设置为默认浏览器。

4. 在工控机上，打开终端，输入以下命令。

   ```bash
   cd ~
   bash ~/user_docker_start_xxx.sh
   ```
   <Aside>
   xxx 为具体版本号。
   </Aside>

5. 上述脚本启动后，Microsoft Edge 浏览器会自动打开 HMI 界面。如未自动打开，请手动在浏览器地址栏输入 `http://127.0.0.1:4090`。

   <Image src={operation_guide_1} alt="" />

   <Aside>
   - `外骨骼设备` 、`VR 控制设备` 和 `主动外骨骼设备` 是三套独立的设备，因此，在切换 `启动外骨骼设备` 、`启动 VR 控制设备` 和 `启动主动外骨骼设备` 时，需要将上一个设备下启动的所有控制一一关闭，即确保 `机器人控制` 区域中的按钮均为蓝色按钮，才可切换设备。
   - 如需关闭 HMI，请先以 Ctrl+C 结束用户端与机器人端程序，再关闭对应终端，最后关闭 Microsoft Edge 浏览器。
   </Aside>
   <Aside type="tip">
      为保证通信，用户端程序约每 10 秒向机器人端发送状态获取请求并更新数据。在两端程序均运行的情况下，仅以下两种场景会导致无法定期获取机器人状态。
      - HMI 界面被关闭：界面被手动关闭且无其他界面时，用户端无法通过浏览器获取机器人端状态。
      - HMI 界面被浏览器后台休眠：多标签页下，若当前视焦未落在 HMI 界面，该界面将被浏览器自动休眠，导致用户端无法发送状态获取请求。
   </Aside>
       

</Steps>

### 启动机器人端程序

<Steps>

1. 在工控机上，打开另一个新终端，通过 SSH 命令远程连接到机器人终端。

   ```bash
   zpf@zpf-MIC-770V3H:~$ ssh robot@192.168.1.100
   # 密码为 robot
   robot@192.168.1.100's password: 
   # 如果终端的用户名以 robot 开头，则表示连接成功。
   robot@ter30jb3-ubuntu:~$ 
   ```

2. 在连接成功的终端中，输入以下命令。

   ```bash
   cd ~
   bash ~/robot_docker_start.sh
   ```
   <Aside>
   如需关闭 HMI，请先以 Ctrl+C 结束用户端与机器人端程序，再关闭对应终端，最后关闭 Microsoft Edge 浏览器。
   </Aside>
</Steps>

在启动 HMI 界面后，便可[使用外骨骼设备](#using-exoskeleton-device)对机器人进行控制。

## 使用外骨骼设备{#using-exoskeleton-device}

### 启动外骨骼控制设备{#start-exoskeleton-device}

import operation_guide_2 from 'img/operation_guide_2.png';
import operation_guide_3 from 'img/operation_guide_3.png';

<Steps>
1. 在[启动 HMI 界面](#start-hmi-interface)后，点击 `启动外骨骼设备` 按钮。启动成功后，按钮将变为红色，同时文字显示为 `停止外骨骼设备`。
   <Image src={operation_guide_2} alt="" />
   <Aside>
   - `外骨骼设备` 、`VR 控制设备` 和 `主动外骨骼设备` 是三套独立的设备，因此，在切换 `启动外骨骼设备` 、`启动 VR 控制设备` 和 `启动主动外骨骼设备` 时，需要将上一个设备下启动的所有控制一一关闭，即确保 `机器人控制` 区域中的按钮均为蓝色按钮，才可切换设备。
   - 当机器人发生故障或机械臂运动异常、不受控制时，请立即点击 `停止外骨骼设备` 按钮，机器人将紧急停止响应。
   </Aside>

2. 弹出 AI2-HMI 窗口。该窗口将实时显示遥操数据，请勿在操作过程中关闭该窗口。

   <Image src={operation_guide_3} alt="" />
   
   <Aside>
   - 若仅有一个显示器，请将该窗口最小化，以确保其不影响 HMI 主界面的操作。若有多个显示器，请将该窗口拖动至非主显示器上，便于观察数据变化。
   - 该窗口不会随 HMI 界面的关闭而自动退出，必须通过点击窗口自身的关闭按钮或 HMI 界面上的 `停止外骨骼设备` 按钮方可将其关闭。
   - 为防止多个窗口同时运行导致显示异常或数据冲突，每次启动前请务必确保仅开启一个 HMI 界面及一个对应的 AI2-HMI 窗口。
   </Aside>

</Steps>

### 启动随动{#start-follow}

import operation_guide_7 from 'img/operation_guide_7.png';
import operation_guide_8 from 'img/operation_guide_8.png';
import operation_guide_9 from 'img/operation_guide_9.png';

<Steps>
1. 在确保完成[启动机器臂](#start-robotic-arm)后，在 `机器人控制` 区域，点击 `启动慢速随动` 按钮。
   <Image src={operation_guide_7} alt="" />

2. 此时页面会弹出如下提示信息，机械臂将慢速运动到遥操臂的姿态。确认该姿态正常后，点击 `确认` 按钮，切换至快速随动。
   <Aside type="tip">
   在慢速随动状态下，机械臂会以较慢的速度跟随遥操臂的运动，便于操作者观察机械臂的运动状态，确认机械臂的运动姿态与遥操臂一致且无异常后，再切换至快速随动，以获得更流畅的操作体验。
   </Aside>

   <Image src={operation_guide_8} alt="" width="450" />

3. 在如下弹窗中，点击 `确认` 按钮，切换至快速随动。
   <Image src={operation_guide_9} alt="" width="450"/>

4. 启动成功后，`启动慢速随动` 蓝色按钮将变为 `停止快速随动` 红色按钮，同时确认 `系统状态` 区域中的随动状态为 `快速随动`。

</Steps>

## 使用手柄

import operation_guide_22 from 'img/operation_guide_22.png';

在[启动外骨骼设备](#start-exoskeleton-device)后，您便可使用手柄对机器人进行控制。

<Steps>

1. 确保手柄接收器已连接至工控机 USB 接口，并且手柄电量充足。

   <Aside>
   - 严禁使用有线方式连接手柄和工控机。
   - 手柄充电规格为 5V/1-2A，不支持快充功能，请勿使用功率超过 66W 的快充充电器。充电状态指示说明如下。
      <Image src={operation_guide_22} alt="" width="450"/>
   </Aside>

   配置方法：长按 HOME 键 3 秒，进入模式切换界面，选择 `安卓模式 -> 接收器` 选项。

3. 在工控机上，打开终端，输入以下命令。

   ```bash
   ros2 run joy joy_node
   ```

   <Aside>
   使用手柄期间，请勿关闭该终端窗口，以免导致手柄无法使用。
   </Aside>

## 采集数据

在[启动外骨骼设备](#start-exoskeleton-device)后，您可以通过 HMI 界面中的 `机器人数据采集` 功能来采集 VR 遥操设备的数据。

<Aside>
`外骨骼设备` 、`VR 控制设备` 和 `主动外骨骼设备` 是三套独立的设备，请勿混用。
</Aside>

在完成数据采集和压缩后，您可以使用 `数据上传` 功能来上传数据。在开始以下操作前，请确保至少已完成一次数据采集和压缩。

<Steps>
1. 点击 `数据上传` 按钮。  
   <Aside>
   在上传过程中，请保持 HMI 界面活跃，且不可切换至浏览器其他标签页，否则可能导致上传失败。
   </Aside>
   <Image src={operation_guide_54} alt="" />

2. 在如下弹窗中，请耐心等待数据上传完成。若需要取消上传，请点击 `取消上传` 按钮。
   <Image src={operation_guide_55} alt="" width="400" />
</Steps>


## 标零

标零操作仅在拆装结构后需要进行。当遥操臂姿态与机械臂姿态不一致，且 AI2-HMI 界面未显示超限时，可能为零位异常，请联系我司售后工程师解决。
