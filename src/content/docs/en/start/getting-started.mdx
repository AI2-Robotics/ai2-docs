Welcome to AI² Robotics products! This guide will help you quickly get started and flexibly operate the robot.

## 🚀 Getting Started

If this is your first time using the robot, start here:

- **Safety Notes**: Please read and strictly follow the [safety regulations](/en/alphabot2/hardware/safety-regulations) and [disclaimer](/en/statement/disclaimer) before use.
- **Hardware Overview**: Refer to the [Hardware Introduction](/en/alphabot2/hardware/architecture) for the structure and specifications of AlphaBot 2.
- **Unboxing and Power-On**: Refer to [Getting Started](/en/alphabot2/hardware/getting-started) for basic operations such as unboxing and powering on.

## 🧭 In-Depth Experience

After mastering basic operations, you can refer to [Common Operations](/en/alphabot2/hardware/operation-guide) to explore more functions and application scenarios of the robot.

## 🎮 Teleoperation

To use the teleoperation function, please refer to [VR Teleoperation](/en/vr-teleoperation/getting-started) or [Exoskeleton Teleoperation](/en/exoskeleton-teleoperation/getting-started) for details, including introduction, operation guides, maintenance and warranty.

## ⚙️ SDK & Custom Development

For custom development, please refer to the [AlphaBotCore SDK](/en/sdk/overview) for detailed development guides and API documentation.
