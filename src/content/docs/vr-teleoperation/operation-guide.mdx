在使用 VR 遥操设备时，掌握以下操作将有助于您更高效地使用和操作设备。

## 启动 HMI 界面

ROBOT-CONTROL-HMI（简称 HMI）是用于控制和监控机器人系统的图形用户界面，也是使用 VR 遥操设备时与机器人交互的核心工具。HMI 分为用户端（user）与机器人端（robot）两部分程序，使用前需分别启动，具体操作如下。

### 启动用户端程序

import operation_guide_1 from 'img/operation_guide_1.png';

<Steps>

1. 在工控机上，下载 [Microsoft Edge 浏览器安装包](https://www.microsoft.com/zh-cn/edge/download?form=MA13FJ) 。

2. 在安装包所在文件夹下，打开终端，通过如下命令进行安装。


   ```bash
   sudo dpkg -i microsoft-edge-stable_xxx_arm64.deb
   ```
   <Aside>
   请将命令中的 `microsoft-edge-stable_xxx_arm64.deb` 替换为实际的安装包文件名。
   </Aside>


3. 在 Ubuntu 操作系统中，依次点击 `设置 -> 默认应用程序 -> 网络(W)`，选择 Microsoft Edge，将 Microsoft Edge 设置为默认浏览器。

4. 在工控机上，打开终端，输入以下命令。

   ```bash
   cd ~
   bash ~/user_docker_start.sh
   ```

5. 上述脚本启动后，Microsoft Edge 浏览器会自动打开 HMI 界面。如未自动打开，请手动在浏览器地址栏输入 `http://127.0.0.1:4090`。

   <Image src={operation_guide_1} alt="" />

   <Aside>
   - `外骨骼设备` 、`VR 控制设备` 和 `主动外骨骼设备` 是三套独立的设备，因此，在切换 `启动外骨骼设备` 、`启动 VR 控制设备` 和 `启动主动外骨骼设备` 时，需要将上一个设备下启动的所有控制一一关闭，即确保 `机器人控制` 区域中的按钮均为蓝色按钮，才可切换设备。
   - 如需关闭 HMI，请先以 Ctrl+C 结束用户端与机器人端程序，再关闭对应终端，最后关闭 Microsoft Edge 浏览器。
   </Aside>
       

</Steps>

### 启动机器人端程序

<Steps>

1. 在工控机上，打开另一个新终端，通过 SSH 命令远程连接到机器人终端。

   ```bash
   zpf@zpf-MIC-770V3H:~$ ssh robot@192.168.1.100
   # 密码为 robot
   robot@192.168.1.100's password: 
   # 如果终端的用户名以 robot 开头，则表示连接成功。
   robot@ter30jb3-ubuntu:~$ 
   ```

2. 在连接成功的终端中，输入以下命令。

   ```bash
   cd ~
   bash ~/robot_docker_start.sh
   ```
   <Aside>
   如需关闭 HMI，请先以 Ctrl+C 结束用户端与机器人端程序，再关闭对应终端，最后关闭 Microsoft Edge 浏览器。
   </Aside>
</Steps>

在启动 HMI 界面后，便可[使用 VR 设备](#using-vr-device)对机器人进行控制。

### 启动 VR 控制设备{#start-vr-control-device}

<Steps>
1. 在确保完成[前置配置](#preconfiguration)后，点击 HMI 界面中的 `启动 VR 控制设备` 按钮。启动成功后，按钮将变为红色，同时文字显示为 `停止 VR 控制设备`。
   <Image src={operation_guide_8} alt="" />

   <Aside>
   - `外骨骼设备` 、`VR 控制设备` 和 `主动外骨骼设备` 是三套独立的设备，因此，在切换 `启动外骨骼设备` 、`启动 VR 控制设备` 和 `启动主动外骨骼设备` 时，需要将上一个设备下启动的所有控制一一关闭，即确保 `机器人控制` 区域中的按钮均为蓝色按钮，才可切换设备。
   - 当机器人发生故障或机械臂运动异常、不受控制时，请立即点击 `停止 VR 控制设备` 按钮，机器人将紧急停止响应。
   </Aside>

2. 查看 `系统状态` 区域，若 `VR 设备状态` 显示为 `数据正常，共 3 条`，则表示启动成功。
</Steps>

## 采集数据

</Aside>
